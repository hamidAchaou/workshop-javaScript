Devine le nombre !

Enoncé

L’utilisateur doit deviner un nombre entre 1 et 50 que le programme a choisi aléatoirement en 10 tentatives. Quand l’utilisateur entre une valeur, on lui affiche le nombre de tentatives restantes, ainsi qu’un message indiquant si sa réponse est correcte ou pas.

Démo

Code source
                    let chances = 10;
                    let count = 0;
                    let random = Math.floor(Math.random() * 50);
                                
                    let rand = document.getElementById("cliko");
                                
                    function enter() {
                      console.log(random);
                      let inpRand = document.getElementById("input-ex4").value;
                      let Tentatives = document.getElementById("ex4-span");
                      let result = document.getElementById("result");
                    
                      if (inpRand <= 50 && inpRand > 0) {
                        
                        while (inpRand != random && chances > 0) {
                        result.innerHTML = "Réesseyez encore !";
                          chances--;
                          count++;
                          Tentatives.innerHTML = `${chances}/10`;
                          break
                        }
                      } 
                      if (inpRand == random) {
                        result.innerHTML = `You won after ${count} Tentatives.`;
                      }
                      if (chances === 0) { 
                        result.innerHTML = "YOU LOST";
                      }
                    }
